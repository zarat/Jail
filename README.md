# Jail - Just Another Interpreted Language

JAIL is a [general purpose scripting language](https://en.wikipedia.org/wiki/General-purpose_language) implementing a subset of [ECMAScript 1](https://www.ecma-international.org/wp-content/uploads/ECMA-262_1st_edition_june_1997.pdf) to integrate in applications and games. Data is exchanged in [JSON](https://www.json.org/json-en.html) format.

[Download](https://github.com/zarat/Jail/releases/)


## Features
 * Dynamically typed [Wiki -> Syntax](https://github.com/zarat/Jail/wiki#syntax)
 * Object oriented
 * extendable in c/c++ [Wiki -> Plugins](https://github.com/zarat/jail/wiki/plugins)
